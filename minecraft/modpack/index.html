Modpack Installation

The modpack can be installed using the Modrinth launcher or Prism Launcher. Other launchers are untested. I personally use Prism Launcher because of its vast set of features, but the Modrinth launcher will also work fine.

Get Prism Launcher

Get the Modrinth App

Using Prism Launcher

1. Go to 'Add Instance' in the top left

2. On the left, go to 'Import'

3. Select 'Browse' and open the modpack (22is5.mrpack) file.

Using the Modrinth App

1. Select the '+' on the left

2. At the top, select 'From File'

3. Click 'Import from file' and select the modpack (22is5.mrpack).
